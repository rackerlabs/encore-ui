(static) initialize(rxActionMenuElement, actionConstructorFnopt) → {rxActionMenu}

Passing in an actionConstructorFn will default to calling that for any calls made to rxActionMenu.action('Action Name'). If it's not defined, it defaults to a function that creates a page object represented by rxActionMenu.action. Most times you'll want to use the default option.
