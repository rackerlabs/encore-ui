Chrome 43+ and Firefox 41+ are required to take advantage of full functionality.
Other browsers and older versions may have degraded or broken functionality. This Element is designed to copy plain text and will not preserve any HTML formatting.

Element to aid in copying plain text to the system clipboard.

Simple Usage

Advanced Usage

You can use the compact property along with the optional .bordered CSS class to apply advanced styling to the Copy Element.

Table Usage

  • Use the .rxCopyTable CSS class on your table to use the Copy Element within table cells. Without this, your table may grow wider than expected.
  • NOTE: The .rxCopyTable will apply a fixed table layout to your table, so there is a high liklihood that you will see a change to your column widths after it is applied.
  • By default, an rxCopy element will be hidden if it is an immediate child of an .rxCopyTable cell.
