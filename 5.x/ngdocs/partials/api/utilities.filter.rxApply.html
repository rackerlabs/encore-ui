rxApply
filter in module utilities

Description

Used to apply an instance of rxSelectFilter to an array.

Merely calls the applyTo() method of a rxSelectFilter instance to an input array. 

<tr ng-repeat="item in list | rxApply:filter">

Usage

In HTML Template Binding

{{ rxApply_expression | rxApply:filter }}

In JavaScript

$filter('rxApply')(list, filter)

Parameters

ParamTypeDetails
listArray

The list to be filtered.

filterObject

An instance of rxSelectFilter
