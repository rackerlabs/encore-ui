rxAppNavItem
directive in module rxApp

Description

Creates a menu item. Recursively creates rx-app-nav if 'children' present. 'Item' must be avialable via scope

Usage

as element:
<rx-app-nav-item>
</rx-app-nav-item>

Directive info

  • This directive creates new scope.

Example

<rx-app-nav-item ng-repeat="item in items"></rx-app-nav-item>
