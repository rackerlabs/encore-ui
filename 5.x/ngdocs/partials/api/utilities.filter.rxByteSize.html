rxByteSize
filter in module utilities

Description

Converts Byte disk size into a more readable format (e.g. MBs, GBs, TBs, PBs)

1000 → 1 KB
12000000 → 12 MB

Usage

In HTML Template Binding

{{ rxByteSize_expression | rxByteSize }}

In JavaScript

$filter('rxByteSize')()
