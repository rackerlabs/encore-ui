rxVisibility
service in module utilities

Description

Provides an interface for adding new visibility methods for nav menus. Methods added via addMethod should have a function (scope, args) interface.

When you do: 

visibility: [ "someMethodName", { foo: 1, bar: 2} ]
in a nav menu definition, the (optional) object will be passed to your method as the second argument args, i.e.: 
function (scope, args) {}
