rxPromiseNotifications
service in module utilities

Description

Manages displaying messages for a promise.

It is a common pattern with API requests that you show a loading message when an action is requested, followed by either a success or failure message depending on the result of the call. rxPromiseNotifications is the service created for this pattern.

Methods

  • add(promise, msgs, stack)

    Parameters
    ParamTypeDetails
    promiseObject

    The promise to attach to for showing success/error messages

    msgsObject

    The messages to display. Can take in HTML/expressions

    PropertyTypeDetails
    loadingString

    Loading message to show while promise is unresolved

    success
    (optional)
    		String

    Success message to show on successful promise resolve

    error
    (optional)
    		String

    Error message to show on promise rejection

    stack
    (optional)
    		String

    What stack to add to

    (default: 'page')

Example

rxPromiseNotifications.add($scope.deferred.promise, {
    loading: 'Loading Message',
    success: 'Success Message',
    error: 'Error Message'
});
