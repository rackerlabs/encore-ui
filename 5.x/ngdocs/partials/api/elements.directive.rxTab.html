rxTab
directive in module elements

Description

Element for creating a tab.

Dependencies

  • rxTabset

Usage

as element:
<rx-tab
       [active="{Boolean}"]
       heading="{String}"
       [on-select="{Boolean}"]
       [on-deselect="{Boolean}"]>
</rx-tab>
as attribute
<ANY rx-tab
     [active="{Boolean}"]
     heading="{String}"
     [on-select="{Boolean}"]
     [on-deselect="{Boolean}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
active
(optional)
Boolean

True to show active tab by default

headingString

Heading text to be displayed by default when rx-tab-heading tags are used

onSelect
(optional)
Boolean

True when tab is selected

onDeselect
(optional)
Boolean

True when tab is deselected
