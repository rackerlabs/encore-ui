rxXor
filter in module utilities

Description

Returns the exclusive or of two arrays.

Usage

In HTML Template Binding

{{ rxXor_expression | rxXor:excluded }}

In JavaScript

$filter('rxXor')(array, excluded)

Parameters

ParamTypeDetails
arrayArray

The first input array

excludedArray

The second input array

Returns

Array
  • A new array of the unique elements in each array.
