rxTypeaheadPopup
directive in module utilities

Description

creates a popup used by the rxTypeahead directive

Usage

as attribute
<ANY rx-typeahead-popup
     [matches]="{string[]}"
     [query]="{string}"
     [active]="{string}"
     [position]="{string}"
     move-in-progress="{boolean}"
     [select]="{string}"
     [assign-is-open]="{boolean}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
[matches]string[]

values matching typeahead query

[query]string

typeahead query made

[active]string

currently selected choice

[position]string

placement of typeahead results

moveInProgressboolean

is typeahead results popup moving?

(default: false)

[select]string

method to execute on completion

[assignIsOpen]boolean

is assignment option open?
