getText() → {Promise.<(String|null)>}

(READ-ONLY) Text value of tooltip (if present).

Warning: This property is known to be unstable in many Selenium end to end test runs in EncoreUI. Returns the tooltip text. Will automatically hover over the tooltip for you to retrieve the text. If there is no tooltip present on hover, returns null.
