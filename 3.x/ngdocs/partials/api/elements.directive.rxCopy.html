rxCopy
directive in module elements

Description

The rxCopy directive is designed to provide programmatic copy-to-clipboard functionality for plain text on the screen.

Dependencies

  • $interpolate
  • $interval
  • $window
  • rxCopyUtil

Usage

as element:
<rx-copy>
</rx-copy>

Directive info

  • This directive creates new scope.

Example

<rx-copy>
  This text will be copied to the system clipboard when you click the
  clipboard icon found nearby on the page.
</rx-copy>
