rxAppSearch
directive in module rxApp

Description

Creates a search input form for navigation

Usage

as element:
<rx-app-search
       [placeholder="{String}"]
       [model="{*}"]
       [submit="{Function}"]>
</rx-app-search>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
placeholder
(optional)
String

Title of page

model
(optional)
*

Model to tie input form to (via ng-model)

submit
(optional)
Function

Function to run on submit (model is passed as only argument to function)
