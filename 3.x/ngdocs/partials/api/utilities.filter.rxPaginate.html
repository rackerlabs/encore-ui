rxPaginate
filter in module utilities

Description

This is the pagination filter that is used to calculate the division in the items list for the paging.

Usage

In HTML Template Binding

{{ rxPaginate_expression | rxPaginate:pager }}

In JavaScript

$filter('rxPaginate')(items, pager)

Parameters

ParamTypeDetails
itemsObject

The list of items that are to be sliced into pages

pagerObject

The instance of the rxPageTracker service. If not specified, a new one will be created.

Returns

Object

The list of items for the current page in the rxPageTracker object
