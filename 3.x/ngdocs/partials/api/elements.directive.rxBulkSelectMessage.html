rxBulkSelectMessage
directive in module elements

Description

This directive is responsible for drawing the appearing/disappearing "message" row in the table header. This row shows how many items have been selected, and gives buttons for "Select All" and "Clear All"

You should not use this directive directly. It will be drawn automatically by rxBulkSelect.

If the table also has rxFloatingHeader available, this directive will communicate with the controller from rxFloatingHeader, to correctly support the appearing/disappearing of this header row.

Dependencies

  • rxBulkSelect

Usage

as attribute
<ANY rx-bulk-select-message
     resource-name="{String}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
resourceNameString

The singular form of the name of the resource, e.g. 'server'.
