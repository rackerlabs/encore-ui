rxSortableColumn
directive in module elements

Description

Renders a clickable link in a table heading which will sort the table by the referenced property in ascending or descending order.

Usage

as element:
<rx-sortable-column
       display-text="{String}"
       sort-method="{Function}"
       sort-property="{String}"
       predicate="{Object}"
       reverse="{Boolean}">
</rx-sortable-column>

Parameters

ParamTypeDetails
displayTextString

The text to be displayed in the link

sortMethodFunction

The sort function to be called when the link is clicked

sortPropertyString

The property on the array to sort by when the link is clicked.

predicateObject

The current property the collection is sorted by.

reverseBoolean

Indicates whether the collection should sort the array in reverse order.
