rxInput
directive in module elements

Description

Structural element directive used for layout of sub-elements. Place your HTML control elements within this directive.

Display:
block (full width of parent)
Parent:
rxFieldContent
Siblings:
Any HTML Element
Children:
  • rxPrefix
  • rxSuffix
  • rxCheckbox
  • rxRadio
  • rxSelect
  • rxToggleSwitch
  • rxOptionTable
  • Any HTML Element

Usage

as element:
<rx-input>
</rx-input>

Example

...
<form rx-form>
  <rx-form-section>
    <rx-field>
      <rx-field-name>Salary:</rx-field-name>
      <rx-field-content>
        <rx-input>
          <input type="number" />
        </rx-input>
      </rx-field-content>
    </rx-field>
  </rx-form-section>
</form>
...
