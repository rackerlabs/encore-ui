rxDiskSize
filter in module utilities

Description

Converts GB disk size into a more readable format (e.g. GBs, TBs, PBs)

420 → 420 GB
125000 → 125 TB
171337000 → 171.337 PB

Usage

In HTML Template Binding

{{ rxDiskSize_expression | rxDiskSize }}

In JavaScript

$filter('rxDiskSize')()
