rxNotification
directive in module elements

Description

Display a static message with styling taken from rx-notifications.

Usage

as element:
<rx-notification
       [type="{String}"]>
</rx-notification>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
type
(optional)
String

The type of notification (e.g. 'warning', 'error')

(default: 'info')

Example

<rx-notification type="warning">This is a message!</rx-notification>
