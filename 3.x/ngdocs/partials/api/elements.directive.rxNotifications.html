rxNotifications
directive in module elements

Description

Displays all messages in a stack.

Usage

as element:
<rx-notifications
       [stack="{String}"]>
</rx-notifications>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
stack
(optional)
String

The message stack to associate with

(default: 'page')

Example

<rx-notifications stack="myCustomStack"></rx-notifications>
