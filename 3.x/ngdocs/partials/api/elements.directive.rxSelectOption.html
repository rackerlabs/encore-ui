rxSelectOption
directive in module elements

Description

A single option for use within rxMultiSelect.

<rx-select-option> is to <rx-multi-select> as <option> is to <select>.

Just like <option>, it has a value attribute and uses the element's content for the label. If the label is not provided, it defaults to a titleized version of value.

<rx-select-option value="DISABLED">Disabled</rx-select-option>

Dependencies

  • rxCheckbox

Usage

as element:
<rx-select-option
       value="{String}">
</rx-select-option>

Parameters

ParamTypeDetails
valueString

The value of the option. If no transcluded content is provided, the value will also be used as the option's text.
