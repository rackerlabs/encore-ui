(static) toMoment(rxAgeString) → {moment}

A moment date object representing the point in time the rxAgeString refers to. This value does not come back as a promise, but is instead a direct value.

Parameters:
Name Type Description
rxAgeString String

An age string, representing the time that has passed since a certain datetime.

// given the current datetime is January 1st, 1970 at noon UTC.
var oneMonthOneDayAgo = new Date('1969-11-31T12:00:00z').valueOf();
expect(encore.rxAge.toMoment('1m 1d').valueOf()).to.equal(oneMonthOneDayAgo)
