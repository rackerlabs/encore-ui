PaginatedItemsSummary
filter in module utilities

Description

Given an active pager (i.e. the result of rxPageTracker.createInstance()), return a string like "26-50 of 500", when on the second page of a list of 500 items, where we are displaying 25 items per page

Dependencies

  • $interpolate

Usage

In HTML Template Binding

{{ PaginatedItemsSummary_expression | PaginatedItemsSummary }}

In JavaScript

$filter('PaginatedItemsSummary')(pager)

Parameters

ParamTypeDetails
pagerObject

The instance of the rxPageTracker service.

Returns

String

The list of page numbers that will be displayed.
