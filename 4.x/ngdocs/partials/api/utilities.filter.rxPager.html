rxPager
filter in module utilities

Description

This is the pagination filter that is used to limit the number of pages shown.

Usage

In HTML Template Binding

{{ rxPager_expression | rxPager }}

In JavaScript

$filter('rxPager')(pager)

Parameters

ParamTypeDetails
pagerObject

The instance of the rxPageTracker service. If not specified, a new one will be created.

Returns

Array

The list of page numbers that will be displayed.
