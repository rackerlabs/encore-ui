rxFormSection
directive in module elements

Description

Structural element directive used for layout of sub-elements.

By default, all rxField, rxSelectFilter, and <div> elements will display inline (horizontally). If you wish to display these elements in a stacked manner, you may place the stacked attribute on rx-form-section.

Display:
block (full width of parent)
Parent:
rxForm
Siblings:
Any HTML Element
Children:
  • rxField
  • rxSelectFilter
  • HTML DIV Element

Usage

as element:
<rx-form-section
       [stacked="{*}"]
       [controlled-width="{*}"]>
</rx-form-section>

Parameters

ParamTypeDetails
stacked
(optional)
*

If present, rxField children will stack vertically rather than display horizontally.

controlled-width
(optional)
*

If present, the element will not consume the full width of its container.

Example

...
<form rx-form>
  <rx-form-section>
    <rx-field>...</rx-field>
    <rx-select-filter>...</rx-select-filter>
    <div>...</div>
  </rx-form-section>
</form>
...
