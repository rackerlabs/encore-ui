rxTokenInterceptor
service in module utilities

Description

Simple $http injector which will intercept http request and inject the Rackspace Identity's token into every http request.

Dependencies

  • rxSession

Example

angular.module('encoreApp', ['encore.ui'])
    .config(function ($httpProvider) {
        $httpProvider.interceptors.push('rxTokenInterceptor');
    });
