rxField
directive in module elements

Description

Structural element directive used for layout of sub-elements.

Display:
block
  • default: shares width equally with sibling rxField and div elements
  • stacked: max-width: 400px
Parent:
rxFormSection
Siblings:
Any HTML Element
Children:
  • rxFieldName
  • rxFieldContent
  • Any HTML Element

Usage

as element:
<rx-field>
</rx-field>

Example

...
<form rx-form>
  <rx-form-section>
    <rx-field>
      <rx-field-name>...</rx-field-name>
      <rx-field-content>...</rx-field-content>
    </rx-field>
  </rx-form-section>
</form>
...
