rxFieldName
directive in module elements

Description

Stylistic element directive that provides a standardized UI for form field names.

Display:
block (full width of parent)
Parent:
rxField
Siblings:
  • rxFieldContent
  • Any HTML Element
Children:
Any HTML Element

Usage

as element:
<rx-field-name
       [ng-required="{Boolean}"]>
</rx-field-name>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ngRequired
(optional)
Boolean

Is this field required? This will add/remove the required symbol to the left of the name.

(default: false)

Example

...
<form rx-form>
  <rx-form-section>
    <rx-field>
      <rx-field-name>Salary</rx-field-name>
      <rx-field-content>...</rx-field-content>
    </rx-field>
  </rx-form-section>
</form>
...
