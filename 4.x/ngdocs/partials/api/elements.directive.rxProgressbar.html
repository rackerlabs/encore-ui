rxProgressbar
directive in module elements

Description

Element used to provide feedback on the progress of a workflow or action.

Usage

as element:
<rx-progressbar
       value="{Expression}"
       [max="{Expression}"]>
</rx-progressbar>

Parameters

ParamTypeDetails
valueExpression

Numeric value used to calculate progress in relation to the max value.

max
(optional)
Expression

Maximum numeric value to calculate progress.

(default: 100)
