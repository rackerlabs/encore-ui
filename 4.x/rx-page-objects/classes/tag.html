isElementPresent

  • isElementPresent(subLocator: Locator): wdpromise.Promise<boolean>

  • Inherited from ElementFinder.isElementPresent

    • Defined in node_modules/protractor/built/element.d.ts:784

    Same as ElementFinder.isPresent(), except this checks whether the element identified by the subLocator is present, rather than the current element finder. i.e. element(by.css('#abc')).element(by.css('#def')).isPresent() is identical to element(by.css('#abc')).isElementPresent(by.css('#def')).

    // Or using the shortcut $() notation instead of element(by.css()):

    $('#abc').$('#def').isPresent() is identical to $('#abc').isElementPresent($('#def')).
